# 用户操作手册

## 系统概述

用户角色权限管理系统提供用户管理、部门管理、角色权限配置、系统配置等功能。

## 快速开始

### 登录

1. 打开系统首页，自动跳转到登录页
2. 输入邮箱和密码
3. 点击"登录"按钮

### 注册

1. 在登录页点击"去注册"
2. 填写邮箱、密码、姓名等信息
3. 点击"注册"按钮

## 功能说明

### 仪表板

登录后进入仪表板，显示欢迎信息和系统状态。

### 用户管理

路径: 左侧导航 → 用户管理

功能:
- **查看用户列表**: 表格展示所有用户
- **创建用户**: 点击"创建用户"按钮，填写信息
- **编辑用户**: 点击用户行右侧"⋯" → 编辑
- **删除用户**: 点击用户行右侧"⋯" → 删除

### 部门管理

路径: 左侧导航 → 部门管理

功能:
- **查看部门树**: 左侧展示部门树形结构
- **创建部门**: 点击"创建部门"按钮
- **编辑部门**: 选择部门后点击"编辑"
- **查看成员**: 选择部门后下方显示成员列表

### 角色管理

路径: 左侧导航 → 角色管理

功能:
- **查看角色列表**: 表格展示所有角色及权限
- **创建角色**: 点击"创建角色"按钮
- **分配权限**: 创建/编辑角色时勾选权限
- **设置数据范围**: ALL/DEPT/SELF/CUSTOM

### 系统配置

路径: 左侧导航 → 系统配置

功能:
- **邮件配置**: SMTP 服务器设置
- **安全策略**: 密码复杂度、登录限制
- **性能配置**: 缓存 TTL、连接池
- **系统参数**: 站点名称、语言

### 个人中心

路径: 左侧导航底部 → 个人中心

功能:
- **编辑资料**: 修改姓名、手机号
- **修改密码**: 需验证旧密码

### 审计日志

路径: 左侧导航 → 审计日志

功能:
- **查看日志**: 所有操作记录
- **筛选**: 按操作类型筛选
- **个人历史**: 底部导航 → 操作历史

## 常见问题

**Q: 忘记密码怎么办？**
A: 联系管理员重置密码。

**Q: 无法登录？**
A: 检查邮箱和密码是否正确，或联系管理员确认账号状态。

**Q: 看不到某些功能？**
A: 联系管理员确认角色权限配置。
